🌱 How I Started

I didn’t grow up coding. In fact, I started with zero tech background. But I’ve always loved designing, storytelling, and turning little ideas into something real.

One day, I stumbled into HTML and CSS — and I was hooked. My first layout was messy, broken, and honestly kind of ugly. But I felt something click. That spark of “I built this” kept me going.

With every bug I fixed, every button I styled, every midnight experiment — I felt like I was planting something. And that’s how my little garden of code began.

Matcha Notes App — React + Vite

Matcha Notes adalah aplikasi pencatatan sederhana yang memungkinkan pengguna membuat, menyimpan, dan menghapus catatan langsung di browser. Dibuat menggunakan React dan Vite, aplikasi ini memiliki antarmuka minimalis dan responsif, serta memanfaatkan localStorage untuk menyimpan data secara lokal tanpa database eksternal.

Bookshelf App — Rak Buku Digital

Bookshelf App adalah aplikasi pengelolaan daftar bacaan yang dibangun menggunakan JavaScript DOM, HTML, dan CSS. Proyek ini merupakan implementasi konsep CRUD (Create, Read, Update, Delete) dan menggunakan localStorage sebagai media penyimpanan data lokal tanpa koneksi ke server. Pengguna dapat menandai buku sebagai sudah dibaca, menghapus, atau memindahkan buku sesuai kebutuhan.

Growth Tracker App

Growth Tracker App adalah aplikasi pelacak perkembangan diri yang dibangun menggunakan HTML, CSS, dan JavaScript DOM. Aplikasi ini memungkinkan pengguna mencatat aktivitas harian, menetapkan tujuan, serta menandai progres secara visual.
